Smart Email Organizer

An AI-powered email management system built for university students. It connects to your Gmail inbox via OAuth 2.0, fetches recent emails, and automatically classifies them into meaningful categories using a hybrid ML approach.

Key Features

🔐
Gmail Integration

Secure OAuth 2.0 authentication to read emails with strict read-only access ensuring your privacy.

🧠
AI Chatbot Assistant

Natural language chatbot powered by Groq (Llama 3.3) that summarizes emails, finds deadlines, and navigates categories.

📈
Adaptive Learning

Users can correct misclassified emails, and the underlying ML model learns from these corrections to improve over time.

📅
Event Calendar

Visual calendar widget showing upcoming events directly extracted from your inbox contents.

Automatic Email Classification Categories

  • 📅 Events — Campus fests, workshops, cultural activities
  • 📚 Academics — Assignments, notices, scholarships, placements
  • 💻 Hackathons — Coding competitions, hackathons, contests
  • 👤 Personal — Friend messages, LinkedIn connections, personal notifications
  • 🗑️ Spam — Promotions, newsletters, automated notifications

How the Classifier Works

The system utilizes a hybrid approach to achieve robust classification accuracy:

1

Keyword Scoring Engine (Primary)

Each category features weighted keyword dictionaries. Email subjects and snippets are scored against these dictionaries. Strong matches receive a weight of 3, while moderate matches receive a weight of 1.

2

Naive Bayes Classifier (Secondary)

A Node.js natural library Bayes classifier trained on seed data behaves as a tiebreaker when keyword scores fall below thresholds.

3

Sender-Based Rules

Known promotional senders are flagged as Spam. Educational senders (e.g., .ac.in, .edu domains) automatically receive an Academics priority boost.

4

User Feedback Loop

When users accurately re-categorize emails, the system retrains the Bayes model and saves sender-level overrides for consistent future sorting.

Setup & Installation

Prerequisites

  • Node.js 18+ installed on your system
  • Google Cloud Console project with Gmail API enabled
  • Groq API key for AI Chatbot functionality

1. Clone and install dependencies

cd smart-email-organizer
npm run install:all

2. Configure environment variables

Create a server/.env file with the following configuration:

GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
GOOGLE_REDIRECT_URI=http://localhost:3001/auth/callback
GROQ_API_KEY=your_groq_api_key
SESSION_SECRET=your_session_secret
PORT=3001
CLIENT_URL=http://localhost:5173

3. Run the application

# Terminal 1 — Start the backend
npm run dev:server

# Terminal 2 — Start the frontend
npm run dev:client

The application will now be securely available at http://localhost:5173.

API Architecture Reference

Method Endpoint Description
GET /auth/google Initiate Google OAuth flow
GET /auth/callback OAuth callback handler
GET /auth/user Fetch the authenticated user context
POST /auth/logout Terminate active session cleanly
GET /api/emails Process and classify all recent inbox items
POST /api/emails/:id/recategorize Trigger ML retraining for category adjustments
POST /api/chat Interact with Context-Aware AI Assistant
